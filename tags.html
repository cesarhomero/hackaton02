LISTADO DE TAGS HTML

header

Define el encabezado de una página

footer

Define el pie de página de una página

main

define el contenido principal de la maquetación de una página

section

define una sección en la página que puede contener artículos

article

define un artículo de página

nav

define una sección que solo contiene enlaces de navegación

ul

define un menú de opciones sin orden

li

define opciones de selección para un menú

aside

define una sección con contenido irrelevante para la página

p

define contenido de texto

div

representa un contenedor genérico

span

representa un contenido de texto sin significado espécifico
